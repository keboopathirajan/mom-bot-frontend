# MoM Bot Frontend

React frontend for the MoM Bot Transcript Service.

## Features
- Microsoft OAuth authentication
- Teams meeting URL input
- Transcript fetching and display
- Dark theme with Material UI
- Download transcripts as text files

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Backend API

This frontend connects to the backend API at:
`https://mom-bot-transcript-service.onrender.com`

## Deployment

Deploy to Vercel, Netlify, or any static hosting service.
